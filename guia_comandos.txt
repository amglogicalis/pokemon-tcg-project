=====================================================
GUÍA DE COMANDOS RÁPIDOS - PROYECTO TCG POKÉMON
=====================================================

--- 1. ARRANQUE DEL PROYECTO (LOCAL) ---

# BACKEND
cd backend                 # Entrar a la carpeta del servidor
npm install                # Instalar dependencias (solo la primera vez o si clonas de nuevo)
npm run dev                # Arrancar en modo desarrollo (con Nodemon si está configurado)
node app.js                # Arrancar de forma simple (si no hay script dev)

# FRONTEND
cd frontend                # Entrar a la carpeta de la interfaz (Vite/React)
npm install                # Instalar dependencias
npm run dev                # Arrancar servidor de desarrollo (suele ser localhost:5173)
npm run build              # Generar archivos para producción (antes de subir a Vercel/Render)

--- 2. CONTROL DE VERSIONES (GIT) ---

# GUARDAR CAMBIOS (DE LOCAL A GITHUB)
git status                 # Ver qué archivos has modificado
git add .                  # Preparar TODOS los archivos para el guardado
git commit -m "mensaje"    # Crear el punto de guardado con una descripción
git push origin main       # Subir tus cambios a la nube (GitHub)

# RECUPERAR / SINCRONIZAR (DE GITHUB A LOCAL)
git pull origin main       # Traer los cambios que haya en GitHub a tu PC

# BOTÓN DEL PÁNICO (DESHACER LÍOS)
git reset --hard HEAD      # Borrar todos los cambios actuales y volver al último commit
git checkout -- archivo.js # Deshacer cambios solo en un archivo específico
git stash                  # Guardar cambios actuales en un "cajón" temporal y dejar todo limpio
git stash pop              # Recuperar esos cambios del "cajón"

--- 3. BASE DE DATOS Y DEPENDENCIAS ---

# INSTALACIÓN DE NUEVAS LIBRERÍAS (EJEMPLOS COMUNES)
npm install axios          # Para hacer peticiones HTTP
npm install dotenv         # Para gestionar variables de entorno (.env)
npm install mongoose       # Para conectar con MongoDB
